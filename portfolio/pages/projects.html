SmartStick 1.8

An innovative assistive device designed to enhance mobility and safety for the visually impaired. This smart walking stick incorporates cutting-edge technology to detect obstacles, provide navigation assistance, and send emergency alerts.

Project Overview

The SmartStick 1.8 was developed to address the challenges faced by visually impaired individuals in navigating their environment independently. The device uses ultrasonic sensors to detect obstacles at different heights and provides haptic feedback to the user.

Key Features

  • Obstacle detection using ultrasonic sensors
  • GPS tracking for location monitoring
  • GSM module for emergency alerts to predefined contacts
  • Haptic feedback system for intuitive user interaction
  • Long-lasting battery with power management system
  • Water-resistant design for all-weather use

Technologies Used

ESP32 Microcontroller Arduino C++ Ultrasonic Sensors SIM800L GSM Module NEO-6M GPS Module Lithium-ion Battery
